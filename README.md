# GeoGeek - Explore the Geographer Inside You

GeoGeek is an interactive web application designed to help users learn about world geography in an engaging way. It provides a platform to explore political territories, discover country details, and view geographical rankings.

## 🌍 Features

- **Map Explorer:** Interactive world map to explore different continents and countries.
- **Country Insights:** Detailed information for each country, including:
  - Capital, largest city, and official languages.
  - Population, area, GDP, and HDI.
  - Currency, time zone, and calling codes.
  - Country flags and brief summaries.
- **Geographical Rankings:** Compare countries based on:
  - Population (most and least populous).
  - Land area (largest and smallest).
  - Population density (most and least populated).
- **Responsive Design:** Optimized for various screen sizes using modern CSS techniques.

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript (ES6+).
- **Libraries:**
  - [jQuery](https://jquery.com/): Simplified DOM manipulation and event handling.
  - [AmCharts (AmMap)](https://www.amcharts.com/): Powering the interactive world map.
  - [TaffyDB](http://taffydb.com/): A JavaScript library for data management.
  - [Font Awesome](https://fontawesome.com/): Vector icons for a modern UI.
- **Reset:** [HTML5 Reset](http://html5reset.org/) for cross-browser consistency.

## 🚀 Getting Started

Since GeoGeek is a static web application, you can run it locally without any complex setup:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/geo-geek.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd geo-geek
   ```
3. **Open `index.html` in your favorite web browser.**
